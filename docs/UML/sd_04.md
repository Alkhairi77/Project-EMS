sequenceDiagram
    autonumber
    actor Org as Organizer / Panitia
    participant View as Attendees List (Blade)
    participant Ctrl as RegistrationController
    participant DB as MongoDB (registrations)

    Org->>View: Cari Nama Peserta & Klik Tombol "Set Hadir (Check In)"
    activate View
    View->>Ctrl: HTTP PATCH /organizer/registrations/{id}/checkin
    deactivate View
    activate Ctrl

    Ctrl->>DB: Fetch Dokumen Registrasi berdasarkan ID
    activate DB
    DB-->>Ctrl: Data Registrasi Terkini
    deactivate DB

    alt Status saat ini sudah 'Checked In'
        Ctrl-->>Org: Return Warning "Tiket Sudah Pernah Dipakai!"
    else Status saat ini masih 'Registered'
        Ctrl->>DB: Update Status Dokumen menjadi 'Checked In'
        activate DB
        DB-->>Ctrl: Sukses Update Status
        deactivate DB
        Ctrl-->>Org: Redirect Back dengan Flash Message Sukses (Badge Hijau)
    end
    deactivate Ctrl
